Box model

  • Content - centrinė dalis
  • Padding - pamušalas, tarpas tarp turinio ir remelio
  • Border - kraštinė
  • Margin - paraštė
  • Position - pozicija, neįtakoja kitų elementų pozicijos
Box model pvz

Content

įtakoja turinio kiekis ir jo nustatymai (pvz: width, height, font size....)

Padding

tarpas tarp turinio ir rėmelio.
Padding gali būti nustatomas 8 būdais

Border

Josta atskirianti padding ir margin.
Su border baigiasi elemento vidinė dalis (ji spalvinama su fonu)

Nustatymai: galima suteikti spalvos linijos storio, tipo, kampų apvalinimo stilius

  • Border: width, style, color
  • border-radius

Margin

gali persidengti, jeigu vienas po kitu elementai esantys yra blokiniai.

super duper (margin: 0 auto; centruoja per centra viso ekrano tik blokinį arba blokini-linijinį )

gali būti taip pat nustatomas 8 būdais

margin iį viršuaus ir apačios neveikia linijiniams elementams!

Position

  • static (default)
  • relative
  • absolute
  • fixed
  • sticky
  • top, right, bottom, left, z-index (ne pozicojos)

Suteikus elementui bet kuria apart static posiciją jis įgauna galimybę naudotis top, rightt, bottom, left z-index nustatymais.
z-index nusako elemento eilės sluoksnį (kaip arti/toli nuo vertotojo elemntas yra, kuris lementas kurį užstoja)
