Grid

Parent nustatymai (Container)

  • display: grid
  • grid-template-rows: [% | px | fr]
  • grid-template-columns: [% | px | fr]
  • grid-template-areas: ........
  • grid-template: [rows] / [columns]
  • gap: row-gap column-gap (atima vietą iš frakcijų)
  • justify-items: start | end | center | stretch (default) - į šonus
  • align-items: start | end | center | stretch (default) | baseline- į viršų ir apačią
  • place-items: align-items justify-items - į viršų ir apačią
  • justify-content: start | end | center | stretch (default) space-arround | space-between | space-evenly - į šonus
  • align-content: start | end | center | stretch (default) | space-arround | space-between | space-evenly - į viršų ir apačią
  • place-content: align-content justify-content
  • grid-auto-columns: [ px | fr | % ] - kaip kurti papildomų elementų stulpelius
  • grid-auto-rows: [ px | fr | % ] - kaip kurti papildomų elementų eilutes
  • grid-auto-flow: row (default) | column | row-dense | column-dense
  • grid: grid-template-rows / grid-template-columns / grid-template-areas / grid-auto-rows / grid-auto-columns / gris-auto-flow (viskas - labai komplikuota)

Children nustatymai (Items)

  • grid-row-start: number
  • grid-column-start: number
  • grid-row-end: number
  • grid-column-end; number
  • grid-row: start / end
  • grid-column: start / end
  • grid-area: name | grid-row-start /grid-column-start / grid-row-end / grid-column-end
  • z-index: number
  • align-self: start | end | center| stretch
  • justify-self: start | end | center| stretch
  • place-self: align-self justify-self
