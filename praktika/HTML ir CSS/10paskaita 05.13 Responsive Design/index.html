Responsive Design

Stilizuoti/išdėstyti elementus ekrane skirtingai, esant skirtingiems ekranų dydžiams.

Breakpoints:

Ekrano lūžio taškas- kai iš vieno išdėstymo (telefoninio dydžio) yra pakeičiamas elementų išdėstymas į kitą išdėstymą (plančetė)

Galima kurti kiek nori (dažniausiai daromi 3) ir ties kokiais nori dydžiais.

  1. x-small- portretiniai telefonai
  2. small - landscape telefonai
  3. medium - tablets
  4. large - laptopai
  5. x-large - desktopai
  6. xx-large - dideli desktopai

kad būtų sukurtas reikia aprašyti jo media querie

Apsirašo taip: @media (min-width: number_px) {...css...}

gali būti @media printers(min-width: xx_px && max-width: xx_px)

media sekeltorius nesuteikia didesnio prioriteto.

Skirtas pritaikyti mobiliems įrenginiams

taip daryti todėl kad yra naudojamas dažniausiai

Todėl pirma rašome stilių jam ir tada kitkam.

Reikai naudoti min-width.
