Form

Formų tagai

  • form
  • label - formos įvesties pavadinimas
  • input - įvesties laukas
  • textarea - įvesties laukas dideliam kiekiui teksto
  • select - opcijų pasirinkties laukas ( nedideliam kiekiui opcijų)
  • datalist - didelio kiekių opcijų pasirinkities laukas
  • option - pasirinkimo laukelis
  • output - formos atsakymo atvaizdavimui laukas

Atributai

  • type - nurodo inputo tipą
    • text - tekstinė įvestis
    • password - slapyvardis (taškėliai)
    • number - skaičius
    • email
    • url - nuoroda(turi prasidėti http
    • checkbox - daugelio pasirinkimas iš daugelio
    • radio - vieno pasirinkimas iš daugelio
    • file - failo pasirinkimas kompiuteryje
    • tel - telefonas
    • submit - formos pateikimas
    • range - skaičiaus pasirinkimas slenkančioje vietoje
    • date - datos pasirinkimas (be laiko)
    • datetime-local - lokalaus laiko pasirinkimas su data
    • month - mėnuo
    • week - savaitė
    • time - laiko pasirinkimas
    • color - spalvos pasirinkimas
    • button - mygtukas be funkcionalumo
    • image - formos pateikimas kaip paveiklsiukas
    • hidden - paslėpta įvestis
    • reset - atnaujinti formą (išvalyti)
    • search - paišką kažkur
  • for - nurodo kuriam elementui priskirtas pagal id
  • id - elemento identifikatorius
  • name - nurodo inputo pavadinimą ( dažniausiai toks kaip id)
  • value - įvesties reikšmė
  • required - nurodo ar privalomas laukas jei yra
  • placeholder - reikšmė esanti prieš užpildant
  • min - mažiausias skaičius
  • max - didžiausias skaičius
  • step - kokiu žingsniu skaičius keičiasi
  • accept - kokio tipo failas priimtinas
  • pattern - kokia įvesties galima seka
  • checked - radio ar checkbox išsankstinis parinkimas
  • rows - text area eilučių kiekis
  • cols - text area stulpelių kiekis
  • selected - select ar datalist pasirinkimas išsankstinis
  • dissabled - išjungtas įvesties laukas (blured)
  • readonly - galima tik matyti kas įvesta
  • multiple - su ctrl galima kelias reikšmes pasirinkti (dažniau checkbox naudoti)

Pavyzdys

Pseudo klasės

pvz css'e

  • :checked - pažymetas checkbox arba radio
  • :default - pradinis su checked atributu
  • :disabled - su tokiu atributu
  • :enabled - su tokiu atributu
  • :focus - ant kurio sufokusuota
  • :in-range - įvestis ribose
  • :out-of-range - įvestis už ribų
  • :indeterminate - nenurodyta reikšmė (sunku padaryt, padaroma per scriptus)
  • :valid - validi reikšmė (pvz email)
  • :invalid - nevalidi reikšmė
  • :optional - įvestis neturinti required
  • :required - įvestis us required
  • :readonly - su readonly attributu
  • :read-write - be readonly attributu

Pseudo elementai

  • ::placeholder - placeholderio parinkimas
  • ::before - turinio parinkimas prieš
  • ::after - turinio parinkimas po
  • ::marker - markeriu parinkimas
  • ::selection - kas pažymėta pasirinkimas
  • ::first-letter - pirma raidė
  • ::first-line - pirma eilutė
  • ::-webkit-scrollbar - scrolbaras (reikia visus modifikuoti kad pasikeistų)
  • ::-webkit-scrollbar-thumb
  • ::-webkit-scrollbar-track
